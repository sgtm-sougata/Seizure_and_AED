Seizure free period data set preparation

  1. Filter data based on the Seizure reported at diagnosis is Yes.
  2. Last Seizure incidence date.
  3. If last seizure incidence date is not known then take date of start of AED.
  4. Last date of follow up if alive if not than taken death date.
  5. Weekly Duration between last date of follow up and last seizure incidence date.
  6. Censored if not alive.
